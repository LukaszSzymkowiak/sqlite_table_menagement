Simple sql tables/columns/rows management created with python and sqlite3
!!!! IN PROGRESS !!!

Functions:
- Main menu:
    - show tables
    - go to table
    - add table
    - remove table
- Table menu:
    - show columns
    - go to column
    - add column
    - remove column
- Column menu:
    - show rows
    - add row to column
    - delete row from column


Technologies:
- python 3.10.6
- sqlite3 3.37.2
